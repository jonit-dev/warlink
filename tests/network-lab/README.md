# Network lab

Phase 0 evidence belongs here. Record:

- Warcraft and ZeroTier versions.
- Operating-system versions and firewall tools.
- Physical network and NAT context without public addresses.
- ZeroTier direct or relayed status.
- Lobby discovery, join, gameplay, and restart behavior.
- Exact temporary firewall changes and their cleanup.

Never commit credentials, network IDs, full node IDs, public IP addresses,
Warcraft files, or personal diagnostic data.
