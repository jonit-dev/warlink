# Security Policy

WarLink manages privileged local services and temporary private-network access.
Please do not report vulnerabilities in a public issue.

Until a private reporting address is published, contact the repository owner
through their GitHub profile and request a private channel. Include the affected
component, reproduction steps, impact, and any suggested mitigation.

Never include real ZeroTier tokens, network IDs, room tokens, host private keys,
Warcraft paths, or unredacted diagnostic bundles in a report.

Only the latest commit on `main` is currently supported. No production release
has been published.
